# IT Landscape
 
Mijn project voor het vak IT Landscape (Graduaat Programmeren, 2025–2026). Een paar opdrachten waarin ik telkens met een andere technologie aan de slag ga.
 
## Opdrachten
 
- **Docker** — basiscommando's, een eigen Dockerfile en Docker Compose
- **CMS** — een CMS opgezet in Docker en er een CV-pagina in gebouwd
- **MongoDB** — een database draaiend gekregen via Docker Compose
- **CI/CD** — een pipeline die automatisch draait met GitHub Actions
## Gebruikt
 
Docker, Docker Compose, MongoDB, Drupal, GitHub Actions, JavaScript
 
## Bestanden
 
| Bestand/Map | Wat erin zit |
|-------------|--------------|
| `Dockerfile` | Docker image definitie |
| `mypage.html` | Webpagina gebruikt bij de CMS-opdracht |
| `.github/workflows/` | CI/CD-pipeline (GitHub Actions) |
